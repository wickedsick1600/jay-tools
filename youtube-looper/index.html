YouTube Looper

Pick a start and end time, and play just that section on repeat. Useful for language practice, music transcription, dance drills, or re-watching one moment.

How to use this

  • Paste a YouTube link (or just the video ID) and click Load.
  • Type a Start and End time, or scrub the video to the moment you want and click Use now.
  • Keep Loop this section on — when playback reaches the end, it jumps back to the start.
  • Turn the loop off to let the video play normally.
  • Share link copies a URL that opens this exact video and loop on any device.

Keyboard shortcuts

  • Space — play / pause
  • [ / ] — set start / end to current playback time
  • L — toggle loop
  • J / K — nudge start earlier / later by 1s
  • , / . — nudge end earlier / later by 1s
  • Home — jump to start

Tired of ads interrupting your loop? Try a browser with a built-in ad blocker like Brave — this tool doesn't (and can't) block YouTube ads itself.

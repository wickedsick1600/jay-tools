How to use this

  • Load your PDF by dropping it onto the box or clicking to pick a file.
  • Pick a page from the page list, then add text, shapes, drawing, or a signature.
  • Use Select to move, resize, or delete items you added.
  • On phones, use Pan when you need to scroll around the page, then switch back to Select or another tool.
  • For signatures, upload once, adjust cleanup if needed, then place it on as many pages as you need.
  • Download when you are done. The output is a new flattened PDF copy.

# folder-tool

Build folder structures visually or from pasted lists, then download as `.zip` and copy clean ASCII trees.

## Local preview

Open `index.html` in a browser. All libraries load from CDN; no build step.

## Deploy

Pushes to `main` auto-deploy to Netlify. No serverless functions — pure static site.

## Input methods

- Visual row builder (folder/file selector + indent/outdent + add child)
- Paste import from Excel/Word/notes
- Sample presets (HR/Admin/Personal)
- Optional plain-list copy for round-trip editing

How to use this

  • Load your image by dropping it onto the box above or clicking to pick a file.
  • Pick a tool from the toolbar: Rectangle, Circle, Line, Text, or Free Draw.
  • Click and drag on the image to draw. Use Select to move, resize, or rotate things you've added.
  • Change color using the color picker on the toolbar.
  • For text, pick a font and size from the text options, then click on the image to place text.
  • Download when you're done — you can resize and choose format/quality first.

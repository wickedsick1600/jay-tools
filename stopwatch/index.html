How to use this

  • Start begins the stopwatch. The big number on top is the total time. The smaller number below is the time since your last split.
  • Split saves the current moment to the list. Type a short note first if you want it attached.
  • Copy buttons let you grab any timecode — just the time, or the time + your note.
  • Reset clears everything.

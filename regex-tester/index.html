How to use this

  • Pattern is the regex — no slashes, just the body (e.g. \d+).
  • Flags: g = find all, i = case-insensitive, m = ^/$ match line starts/ends, s = dot matches newline.
  • Matches highlight live as you type. Capture groups show in the list below.

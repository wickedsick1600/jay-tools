Privacy — Prompt Enhancer

Last updated: May 8, 2026

Important: Unlike the other tools on this site, this one does send your text to a third-party AI provider. Don't paste anything confidential, private, or sensitive into it.

What happens when you click "Enhance prompt"

  1. Your text and the AI tool you picked (like "ChatGPT" or "Midjourney") are sent to a small server function on this site.
  2. That server function passes your text to an AI API (by default OpenAI, unless the site operator configured a compatible service such as Groq) using a secret key that stays on the server.
  3. The provider returns a polished prompt, which is sent back to your browser.
  4. Nothing is stored on my side. Each request is independent — no history, no account.

What the AI provider might do with your text

Processing is governed by that provider's policies. For OpenAI, see OpenAI's privacy policy and terms of use. If the operator uses a different API, that provider's terms apply instead. Don't submit anything confidential, personal, or proprietary.

Fair-use limit

To stop one person from using up the shared AI quota, the server keeps a short in-memory count of how many times each IP address has called the tool. That count lives in the server's memory only and is wiped whenever the server restarts.

What I don't do

  • No accounts, no logins, no email collection.
  • No tracking scripts, no analytics, no ads.
  • No storing of your prompts or the enhanced output.

Server logs

Netlify, the company that hosts this page, automatically records standard server logs whenever anyone loads the site (IP address, time, browser type). This is normal for any website.

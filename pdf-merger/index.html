PDF Merger

Combine multiple PDFs into one file without uploading anything. Put the files in the exact order you want before downloading.

Your PDFs are never uploaded. They are merged in your browser.

Merge order

    Add at least 2 PDF files. The first file in this list becomes the first pages in the merged PDF.

    How to use this

    • Add PDFs using the file picker.
    • Set the order with Move up / Move down, or drag files on desktop.
    • Remove anything unwanted before merging.
    • Download the merged PDF when the order looks right.
